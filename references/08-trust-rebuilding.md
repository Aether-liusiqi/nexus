# 信任重建路线图（四疗法融合）

## 前置评估：信任可以重建吗？

不是所有的信任破裂都能修复。以下三个条件全部满足，才有重建的可能：

1. **伤害方承认伤害，不辩护。** 如果还在说"我出轨是因为你……"——重建还没开始。
2. **受害方的核心安全感没有完全清零。** 如果听到对方手机响就会惊恐发作——需要先稳定，再谈修复。
3. **双方都愿意投入修复过程。** 单方面修复信任是不可能的。如果一方已经在心理上完全离开了，另一方需要面对这个现实。

**如果以上 3 条任何一条不满足，先做辨识咨询（见 examples/07 或 17），而不是信任重建。**

---

## Phase 1：停止伤害（第 1-4 周）

信任重建的第一步不是"建立信任"，是**停止继续破坏它**。

### 伤害方的行动清单

- [ ] 完全、立即、永久切断与第三方的所有联系。如有必要展示行为证据
- [ ] 同意透明度措施——主动分享，不等待被发现
  - 手机密码、社交账户访问权限（临时性、有边界地，由双方约定范围）
  - 行踪主动报备——不是"被查岗"，是主动说"我到公司了"
- [ ] 接受受害方在这一阶段的情绪波动——愤怒/哭泣/质问/"你和他到底……"
  - 这不是对方"翻旧账"，是依恋创伤的自然反应
  - 回应方式：不辩护、不反问、不反过来指责
  - "我听到了。我做了伤害你的事。我现在在这里。"
- [ ] 不要求对方"赶紧翻篇"——创伤修复没有加速器

### 受害方的行动清单

- [ ] 允许自己感受愤怒和悲伤——不压抑
- [ ] 给自己设定每天"反复思考这件事"的时间上限——比如每天 30 分钟，其余时间把注意力拉回到当下
- [ ] 不把伤害方的过往错误作为后续每一次冲突的武器

### 双方的承诺

- [ ] 不要求对方马上原谅
- [ ] 如果双方有孩子——不让孩子卷入

## Phase 2：一致性验证（第 2-6 个月）

Phase 1 说"我会改"。Phase 2 是"让我看到你确实改了。"

这一阶段的核心原则：**信任不是被说服出来的，是在反复的、一致的行为中被证实的。** 一次失信，计时器重置。

### 伤害方的行动清单

- [ ] 所有承诺的行动全部兑现。无一例外。没有"忘了"。没有"下次一定"
- [ ] 当对方需要确认时——即使已经确认过 50 次——耐心回应，不防御
- [ ] 如果出现失误——立即承认，不找借口："我搞砸了。对不起。我会……"
- [ ] 开始每周一次"检查对话"——（见下方模板）

### 检查对话模板

> **每周一次，15 分钟。** 固定时间（如周日晚饭后）。
>
> 三个问题：
> 1. 这一周在信任方面，你有什么不安或疑惑吗？
> 2. 我做了什么让你感到安全了吗？
> 3. 下周有什么我可以做得更好的？

### 受害方的行动清单

- [ ] 注意并承认对方做的积极行动——即使很小
- [ ] 区分"对方现在的行为"和"过去的伤害"——当对方做了对的行动时，不让过去的创伤完全覆盖当前的体验
- [ ] 如果情绪反复是正常的——但如果情绪完全主宰了日常生活，需要考虑寻求个人心理咨询

## Phase 3：依恋创伤修复（持续）

信任破裂的核心不只是"你做了不该做的事"——是**"我以为你是我的安全基地，但你成了伤害的来源。"** 这需要依恋层面的修复，而不仅仅是行为层面的纠正。

### 依恋创伤修复对话

**来源：EFT + Imago**

#### 步骤 1：伤害方倾听而不辩护

受害方说出被伤害的感受——不是为了让对方难受，而是为了让自己的痛苦被见证。

伤害方的任务：听。不说"但是"。不说"那是过去的事了"。不说"你也有……"。听完后说："我听到了。谢谢你告诉我。"

#### 步骤 2：伤害方用依恋语言回应

不是"我错了，行了吧"——是依恋语言：

> "我让你感到——在最需要我的时候——我不在。我不在安全基地上。我让你一个人面对了最深的恐惧。对不起。"

#### 步骤 3：受害方冒险表达依恋需求

> "我需要知道——当我最脆弱的时候——你会在。"

这是信任重建中最脆弱也最关键的一步。如果这一步被尊重地接住了，信任修复的核心就发生了。

#### 步骤 4：共同书写新叙事

不是"我们经历了出轨"，而是"我们经历了破裂，也正在经历修复。"

---

## Phase 3 的里程碑检查点

| 检查点 | 信号 |
|--------|------|
| ✓ 受害方不再需要每天检查伤害方的行踪 | 安全感开始内化 |
| ✓ 伤害方在被询问时不感到防御 | 透明度已成为习惯 |
| ✓ "检查对话"从每周一次变为每两周一次 | 双方信任自主运转 |
| ✓ 提到过去事件时不再有心率反应 | 创伤记忆开始整合 |

---

## 何时承认修复失败？

如果以下情况持续存在超过 3 个月：
- 一方持续无法原谅（每天愤怒、每场冲突都拉回到出轨事件）
- 一方在 Phase 2 内再次失信
- 双方的"检查对话"从未开始或每次都以争吵收场

——那么信任重建可能已经超出了自我修复的范围，需要专业的伴侣治疗师介入，或者需要面对"分开"的现实。
